Place 32-bit TGA textures here (power-of-two sizes):

  orb_bg.tga     128 x 128   (dark radial gradient, transparent edges)
  orb_border.tga 128 x 128   (thin ring, highlight/shadow)
  glow_soft.tga  256 x 256   (soft bloom, mostly transparent)
  ring_runes.tga 512 x 512   (faint runic ring)

Paths are loaded via ShammyTime_Media.lua. Use /wfassets in-game to test.
